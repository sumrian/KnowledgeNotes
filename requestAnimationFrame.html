requestAnimationFrame 时间戳参数

理解DOMHighResTimeStamp参数如何优化动画性能

时间戳参数的作用

传给 requestAnimationFrame() 的回调函数接收一个 DOMHighResTimeStamp 参数，该参数表示浏览器计划进行下一次重绘的时间点。

这个时间戳非常重要，因为它使开发者能够：

  • 计算自上一帧以来经过的时间（delta time）
  • 根据实际经过的时间更新动画状态
  • 确保动画速度与帧率无关
  • 避免不必要的计算和重绘
  • 实现更平滑、更一致的动画效果

对比：使用 vs 未使用时间戳

未使用时间戳： 动画基于固定的帧率假设。如果实际帧率下降，动画会变慢。

使用时间戳： 动画基于实际经过的时间。无论帧率如何变化，动画速度保持一致。

代码示例

let lastTime = 0;
function animate(timestamp) {
  // 计算自上一帧以来的时间差
  const deltaTime = timestamp - lastTime;
  lastTime = timestamp;

  // 基于时间差更新位置
  const distance = (speed * deltaTime) / 1000;
  element.style.left = `${position + distance}px`;

  requestAnimationFrame(animate);
}
requestAnimationFrame(animate);
