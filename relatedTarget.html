relatedTarget 属性说明

在 mouseover 事件中，relatedTarget 表示鼠标移出的元素。

在 mouseout 事件中，relatedTarget 表示鼠标移入的元素。

对于其他事件类型，relatedTarget 的值总是 null。

// 事件监听器示例
element.addEventListener('mouseover', function(event) {
  // 鼠标移入时，relatedTarget 是鼠标来自的元素
  console.log('来自: ', event.relatedTarget);
});

element.addEventListener('mouseout', function(event) {
  // 鼠标移出时，relatedTarget 是鼠标前往的元素
  console.log('前往: ', event.relatedTarget);
});

其他事件中的 relatedTarget

对于非 mouseover/mouseout 的事件，relatedTarget 属性总是返回 null。

尝试点击上面的按钮触发不同类型的事件，观察日志中 relatedTarget 的值。

// 对于其他事件类型
element.addEventListener('click', function(event) {
  console.log(event.relatedTarget); // 总是 null
});
