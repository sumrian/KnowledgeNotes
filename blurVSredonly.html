技术说明与比较

readonly 属性

✓ 语义化实现

✓ 允许选中和复制内容

✓ 键盘导航友好

✓ 现代浏览器标准

blur() 方法

➤ 传统浏览器兼容方案

➤ 无法选中或复制内容

➤ 键盘导航可能被打断

➤ 主要用于特殊情况

// 使用readonly属性
document.getElementById('username').readOnly = true;

// 使用blur()方法模拟只读
document.getElementById('old-username').addEventListener('focus', function() {
  this.blur(); // 立即移除焦点
});

在现代Web开发中，优先使用readonly属性创建只读字段，因为它提供了更好的用户体验和可访问性。blur()方法主要用于特殊场景，如防止用户聚焦某些元素或在特定条件下移除焦点。
