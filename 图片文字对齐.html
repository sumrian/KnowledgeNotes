图片与文字中线对齐方案

默认情况下，图片和文字在水平排列时是按照基线(baseline)对齐的。 使用Flex布局的align-items: center属性可以轻松实现中线对齐。 下面演示不同对齐方式的效果。

中线对齐效果

使用CSS Flexbox实现图片与文字的中线对齐

美丽的自然景观

这是一段描述图片的文字内容，用于演示图片与文字的中线对齐效果。中线对齐使整体布局更加平衡和美观。

Flexbox的align-items属性可以轻松控制垂直方向上的对齐方式。

海洋之美

另一组图片与文字的组合，展示中线对齐的效果。无论图片高度如何变化，中线对齐都能保持良好的视觉平衡。

尝试使用上方的按钮切换不同的对齐方式，体验它们之间的差异。

实现代码

.image-text-container {
    display: flex;
    align-items: center; /* 关键属性：实现中线对齐 */
    gap: 20px;
}

/* 图片容器样式 */
.image-wrapper {
    width: 120px;
    height: 120px;
    flex-shrink: 0; /* 防止图片被压缩 */
}

/* 图片样式 */
.image-wrapper img {
    width: 100%;
    height: 100%;
    object-fit: cover;
    border-radius: 8px;
}
